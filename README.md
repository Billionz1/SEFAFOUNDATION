## Quick Deploy
### Netlify Drop
1. Drag `sefa-foundation-site/` to [netlify.com/drop](https://app.netlify.com/drop)

### GitHub Pages
1. Push to GitHub
2. Settings → Pages → Deploy from main branch

### cPanel
1. cPanel → Git Version Control → Add GitHub repo to `public_html`

## PayPal Setup
1. PayPal Business → Tools → PayPal Buttons → Donate
2. Replace `YOUR_HOSTED_BUTTON_ID` in `js/script.js`
3. Redeploy
